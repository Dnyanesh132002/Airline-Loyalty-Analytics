# Airline Loyalty Analytics Dashboard

## Project Overview

This project analyzes customer loyalty behavior and flight activity data to understand customer engagement, loyalty program performance, and customer value. The project was completed using Python, SQL, and Power BI.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQL
* Power BI
* DAX

## Project Workflow

* Collected customer loyalty and flight activity datasets.
* Performed data cleaning and preprocessing using Python.
* Handled missing values and corrected data quality issues.
* Conducted exploratory data analysis using Python visualizations.
* Performed SQL analysis to answer business questions.
* Created KPIs and measures using DAX.
* Designed an interactive Power BI dashboard.

## Dashboard Page 1: Executive Overview

This page provides a high-level overview of airline loyalty program performance.

### KPIs

* Total Customers
* Average Customer Lifetime Value (CLV)
* Total Flights
* Average Salary
* Redemption Rate

### Visuals

* Monthly Flight Trend
* Points Accumulated vs Redeemed Trend
* Flights by Province
* Loyalty Card Distribution
* Flight Activity Analysis

## Dashboard Page 2: Customer Analytics

This page focuses on customer demographics and customer value analysis.

### KPIs

* Total Customers
* Average Salary
* Average CLV
* Cancellation Rate

### Visuals

* Customer Distribution Across Canada
* Customer Education Profile
* Gender Distribution of Customers
* Average CLV by Loyalty Card Type
* Customer Enrollment Trend
* Salary vs Customer Lifetime Value (CLV)

## Key Insights

* Ontario has the largest customer base.
* Aurora loyalty card holders generate higher customer value.
* Customer enrollment increased over time.
* Higher-income customers generally have higher CLV.
* Loyalty point accumulation is significantly higher than redemption activity.
